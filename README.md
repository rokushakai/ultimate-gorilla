# 究極ゴリラ ULTIMATE GORILLA

究極のUMA「究極ゴリラ」を探す勇者の子孫たちの旅。
**スマホ1画面で延々と経験値を稼ぐ、UMA収集ブラウザRPG**です。

HTML / CSS / JavaScript のみで作られており、外部ライブラリやビルド、
サーバーの起動は一切不要です。ブラウザで `index.html` を開くだけで
そのまま動作します。

🎮 **公開URL（GitHub Pages）:** https://rokushakai.github.io/ultimate-gorilla/
*( 公開直後はビルドに数分かかることがあります )*


## 🎯 目的

**伝説のUMA「究極ゴリラ」を捕まえよう！** そのために、フィールドを歩いてUMAを探し、
倒す/逃げる/捕まえることで経験値を得てレベルを上げ、装備を集め、UMA図鑑を埋めながら、
最終的に究極ゴリラに挑むのがこのゲームの目標です。
(初回プレイ時にもこの目的説明が表示されます。⚙️設定の「🎯目的を見る」からいつでも再確認できます)


## 概要

- 育つのは主人公だけ。UMA は **育成せず、捕まえて図鑑に登録する・売る対象**として扱う収集RPGです。
- フィールドを歩き、ランダムエンカウントで戦闘 → 経験値でレベルアップ → まほうを習得、というループを1画面の中で繰り返します。
- 商人で武器やアイテムを売買し、神社で職業（部活）を変えて戦い方を調整できます。
- 最終目標は、ラスボス級の伝説のUMA「究極ゴリラ」🦍 を**発見すること**です。


## 遊び方

1. フィールド画面で草原（緑のタイル）を歩くと、一定確率でモンスターやUMAに遭遇します。
2. 戦闘では「⚔ たたかう」「✨ まほう」「🧪 アイテム」「🪤 つかまえる」「💨 にげる」から選びます。
3. 倒す・逃げる・捕まえる、いずれの結果でも経験値が入りレベルアップします。
4. レベルアップ時にランダムで新しい「まほう」を1つ習得します。
5. 村にある 🏪 商人で武器/やくそう/捕獲ロープを売買し、UMAを売ってお金にできます。
6. 村にある ⛩️ 神社で職業（部活）を変更でき、ステータスや得意分野が変わります。
7. レアUMA（ビッグフット・ネッシー・イエティ・ジャージーデビルなど）はレベルに関係なく低確率で出現します。強いので、勝てない時は「にげる」を選びましょう。
8. ごく低確率で伝説のUMA「究極ゴリラ」が出現します。これを発見することがゲームの最終目標です。
   **究極ゴリラはラスボス級で、レベル99・最強装備クラスでもほぼ倒せません。** 出会ったら「にげる」でほぼ確実に逃げられます。安心して発見を楽しんでください。
9. 敵のHPを0にすると「○○に逃げられた！」という演出になり、経験値とお金が手に入ります。**このゲームは「倒す」ではなく「弱らせて捕まえる/逃げられる」UMA収集RPGです。**
10. HPを減らすほど捕まえやすくなります。1〜10程度まで弱らせると捕獲成功率がかなり上がります。
11. 低確率で **メタルゴリラ系** (メタルゴリラ/はぐれメタルゴリラ/フルメタルゴリラ)が出現します。防御が高く攻撃はほぼ通らない代わりに、倒すと高い経験値が手に入ります。
9. スタート地点の 🏠 **実家**に戻ると、「休む」を選んでHP・MP・状態異常を全回復できます。セーブもされます。
10. フィールドに 🎁 **宝箱**が4箇所あります。触れると中身(お金・回復アイテム・装備)をランダムで入手できます。一度開けた宝箱(📦)は空になります。
11. 商人や宝箱でコーヒー/パン/お弁当/ラーメンなどの**回復アイテム**を入手できます。戦闘中の「🧪アイテム」または画面上部の「🎒アイテム」から使えます(HP満タン時は消費されません)。
12. 画面上部「🎒アイテム」ボタンで、**フィールド上でもアイテムを使用**できます。やくそう・コーヒー等でHP回復、せき止めシロップ・デオドラントスプレーで状態異常治療が可能です。
13. 画面右上「🎽装備」で、**所持している**武器・防具・盾・兜をいつでも変更できます。
14. 一部の敵の攻撃で「アレルギー」「におい」などの状態異常になることがあります。時間で自然に治るほか、せき止めシロップ/デオドラントスプレーで治療できます(治療アイテムは商人の「買う」や宝箱で入手できます)。
15. 村の 🍺 **酒場**に入ると旅の仲間を加入させられます。仲間はパッシブ効果で主人公を
    サポートします(詳しくは「仲間の増やし方」を参照)。

### 装備の入手と変更・売却

最初は木の棒/Tシャツ/段ボールのたて/男塾ハチマキだけを所持しています。
村の商人で「⚔武器を買う」「🥋防具を買う」「🛡盾を買う」「⛑兜を買う」から
新しい装備を購入すると所持品に追加され、画面右上「🎽装備」の装備変更画面で
選べるようになります(購入しただけでは装備されないので、装備変更画面で
改めて装備してください)。所持していない装備は「未所持」と表示され選べません。

商人の「🔧装備を売る」から所持装備を売却できます(売値は購入価格の半額)。
**現在装備中のアイテムは売れません**。不要になった装備を整理するのに使えます。

### 宝箱の探し方と中身

フィールドの草原に 🎁 宝箱が4箇所散らばっています。プレイヤーが触れると
中身(お金・コーヒー/パン/お弁当などの回復アイテム・低〜中ランクの装備)を
ランダムで入手できます。一度開けた宝箱は 📦 に変わり、同じデータでは再入手
できません。装備はすでに持っている場合、代わりにゴールドを獲得します。

### フィールド上でのアイテム使用

画面上部の「🎒アイテム」ボタンを押すと、フィールド上でもアイテムを使えます。

- **回復**: やくそう・コーヒー・パン・お弁当・ラーメンでHPを回復
- **治療**: せき止めシロップ(アレルギー)・デオドラントスプレー(におい)で状態異常を治療
- HP満タン時や対応する状態異常がない時は消費されません

戦闘中は「🧪アイテム」から同じアイテムを使えます。

### 仲間の増やし方

村の 🍺 酒場に入ると「仲間を探す / 仲間を見る / 仲間を外す」の3メニューが開きます。

「仲間を探す」から4人の仲間候補を確認し、「加入」を押すと仲間になります。
仲間はUMAと異なりパーティーメンバーとして主人公をサポートします。

| 仲間 | 特色 | 効果 |
|---|---|---|
| 💪 ジュリタニ | 会心の一撃の確率が高い | 攻撃時、低確率で会心の一撃(ダメージ×1.5) |
| 🪤 シュリタニ | UMAを捕まえるのが得意 | 捕獲率+10% |
| 💨 ノリオ | 逃げるのがうまい | 逃走成功率+15% |
| ✨ ハルミ | まほうが得意 | まほうの効果+20% |

- 仲間は最大**2人**まで連れていけます。上限に達した場合は加入できません。
- 「仲間を外す」でいつでも外せ、再度「仲間を探す」から再加入できます。
- 仲間情報はセーブデータに保存され、リロード後も維持されます。
- ニューゲームを行うと仲間情報もリセットされます。

### 実家での回復とセーブ

スタート地点の 🏠 に戻ると「実家」イベントが発生します。「休む」を選ぶと
HP・MP・状態異常が全回復し、その場でセーブされます。危なくなったら実家に
帰って回復しましょう。

行動するたびに自動でセーブされます(オートセーブ)。⚙️設定モーダルにも
その旨が表示されており、「💾今すぐセーブ」ボタンでいつでも手動セーブも
できます。設定モーダル最下部の「🔄ニューゲーム」ボタンでセーブデータを
リセットして最初からやり直すこともできます(確認ダイアログあり)。

より詳しい遊び方やトラブルシュートは [README.txt](README.txt) に、
ゲームの仕様・企画意図は [GAME_DESIGN.md](GAME_DESIGN.md) にまとめています。


## 操作方法

| 操作 | 内容 |
|---|---|
| 画面下の十字キー（▲▼◀▶） | タップで1歩移動。押しっぱなしで連続移動（速度は設定で変更可） |
| フィールドのスワイプ | 上下左右にスワイプして移動（スマホ向け） |
| キーボードの矢印キー | PCでの動作確認用 |
| 画面右上「📖図鑑」 | これまでに遭遇/捕獲したUMAの一覧を表示 |
| 画面右上「🎽装備」 | 所持している武器・防具・盾・兜を変更 |
| 画面右上「📊ステータス」 | レベル/HP・MP/所持金/職業/装備/まほう/所持アイテム/図鑑進捗などを確認 |
| 画面右上「⚙️設定」 | 歩く速さの変更、セーブ、目的の再確認、ヘルプを開く |

スマホ縦画面を前提にしていますが、PCブラウザでも幅480px程度の
中央レイアウトで見やすく表示されます。


## ZIPで配布する方法

Node.js / Python なしで、`index.html` をダブルクリックするだけで
動作します（完全オフライン対応）。

1. このリポジトリを `Code → Download ZIP` でダウンロード、または
   `git clone` してフォルダを取得します。
2. フォルダ全体（`index.html` / `style.css` / `script.js` / それ以外のREADME等）を
   右クリック → 「送る」→「圧縮(zip形式)フォルダー」などでZIP化します。
3. 相手はZIPを解凍して `index.html` を開くだけで遊べます。
   インストール作業やインターネット接続は不要です。


## GitHub Pages 公開手順

このリポジトリでは既にGitHub Pagesを有効化済みです
（公開URLは冒頭に記載）。同じ設定をGitHub上のUIから確認・変更したい場合は、
リポジトリの **Settings → Pages** を開き、以下の内容になっていることを確認してください。

1. `Settings` タブを開く
2. 左メニューの `Pages` を開く
3. `Source` が `Deploy from a branch` になっている
4. `Branch` が `main` になっている
5. `Folder` が `/ (root)` になっている
6. （変更した場合は）`Save` する
7. 数分後、ページ上部に公開URL（`https://<ユーザー名>.github.io/ultimate-gorilla/`）が表示されます

別のアカウント/リポジトリで初めて公開する場合は、上記を新規に設定し、
公開後にこのREADME.md冒頭の「公開URL」欄を実際のURLに書き換えてください。

`index.html` はリポジトリ直下にあり、`style.css` / `script.js` は
すべて相対パスで読み込んでいるため、サブディレクトリでホストされる
GitHub Pages でもそのまま動作します。


## GitHubへのアップロード手順（参考: 別環境にリポジトリを作り直す場合）

このリポジトリは既に `https://github.com/rokushakai/ultimate-gorilla` に
push済みで、GitHub Pagesも有効化されています。別のPCやアカウントで
同じ手順を再現したい場合は、以下のコマンドで作成できます。

```bash
git init
git add .
git commit -m "Initial playable prototype"
git branch -M main
git remote add origin https://github.com/<ユーザー名>/ultimate-gorilla.git
git push -u origin main
```

※ 初回pushでGitHubの認証（ブラウザでのログイン、または
  Personal Access Token の入力）を求められる場合があります。
  画面の指示に従ってください。
※ `gh` CLI（GitHub CLI）でログイン済みであれば、
  `gh repo create ultimate-gorilla --public --source=. --remote=origin` で
  リポジトリ作成とリモート設定を一度に行うこともできます。


## 今後追加予定の機能

- **仲間システム**: フィールド上のNPCに話しかけると、仲間になる/断られる/
  戦闘になる、のいずれかが発生します。仲間になると一緒に歩き、固有の能力を
  持つようになる予定です(企画段階・未実装)。
- 宝箱の増設・レアドロップ拡充(最強装備の入手ルートとしても活用予定)
- 酒場での仲間加入(仲間システムと連携)
- 複数の街・ダンジョン・ワールドマップ間の移動（現在は1枚のマップのみ）
- BGM/SE の追加
- ストーリー性のあるイベント・NPCとの会話
- セーブデータにプレイヤー座標やフィールドアイテムの取得状況も含める

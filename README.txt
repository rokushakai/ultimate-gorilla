====================================================
究極ゴリラ ULTIMATE GORILLA (試作版 v2)
====================================================

究極のUMA「究極ゴリラ」を探す勇者の子孫たちの旅。
HTML / CSS / JavaScript のみで作られたブラウザRPGです。

このゲームは「UMA収集RPG」です。育つのは主人公だけで、
UMAは育成せず、捕まえて図鑑に登録する・売ってお金にする対象として
扱っています(ポケモンのように仲間にして戦わせる育成要素はありません)。

Node.js や Python、サーバーの起動は一切不要です。
インターネット接続も不要です(完全オフラインで動作します)。


----------------------------------------------------
■ 起動方法
----------------------------------------------------
1. このフォルダを好きな場所に展開(解凍)してください。
2. フォルダの中にある index.html をダブルクリックしてください。
   → 既定のブラウザ(Google Chrome や Microsoft Edge)が開き、
     そのままゲームが起動します。

  ※ ファイル構成は以下の通りです。すべて同じフォルダに
    入っている必要があります。
      index.html  … ゲーム画面の本体
      style.css   … 見た目のスタイル
      script.js   … ゲームのロジック・データ(プログラム本体)
      README.txt  … この説明書

  ※ 推奨ブラウザ: 最新版の Google Chrome または Microsoft Edge
    (Windows 11 に標準で入っている Edge でそのまま動作します)

  ※ スマホで遊びたい場合は、このフォルダをZIPにしてスマホに
    送り、解凍してから index.html をブラウザで開いてください。
    (Android の Chrome、iPhone の Safari どちらでも動作します)


----------------------------------------------------
■ 配布方法(ZIP化)
----------------------------------------------------
このフォルダ全体(index.html / style.css / script.js / README.txt)を
右クリック→「送る」→「圧縮(zip形式)フォルダー」などでZIP化し、
相手に渡してください。相手は解凍して index.html を開くだけで遊べます。
インストール作業は不要です。

  ※ backup_v1 フォルダは旧バージョンの控えです。配布時に含めても
    問題はありませんが、不要であれば削除しても構いません。


----------------------------------------------------
■ 遊び方
----------------------------------------------------
[フィールド画面]
・画面上のマップを移動します。
  - スマホ: 画面下の十字キー(▲▼◀▶)をタップ、または
            フィールド画面を上下左右にスワイプ
  - PC:    キーボードの矢印キーでも移動できます(動作確認用)
・草原(緑のタイル 🟩)を歩くと、一定確率でモンスターや
  UMA(未確認生物)に遭遇します。
  茶色の道(🟫)や村(🏠)、商人(🏪)、神社(⛩️)、酒場(🍺)の上では
  遭遇しません。
・マップに落ちている 🗡️(武器)に触れると自動で拾い、
  こうげき力が上がります。
  🧪(やくそう)に触れると自動で拾い、戦闘中に使えます。
・スタート地点の 🏠 実家に戻ると「実家に帰ってきた。」イベントが発生します。
  「休む」を選ぶとHP・MP・状態異常がすべて全回復し、自動セーブされます。
  旅で危なくなったら実家に帰って休みましょう。
・初回起動時のみ、王様からのオープニングメッセージと、続けて
  「目的」の説明が表示されます(一度見ると次回以降は自動表示
  されませんが、⚙️設定の「🎯目的を見る」からいつでも見直せます)。
・村の中にある 🍺(酒場)に触れると酒場メニューが開きます。
  「仲間を探す / 仲間を見る / 仲間を外す」から仲間候補4人に
  声をかけられます。断られても何度でも再挑戦できます。
  詳細は後述の「仲間について」を参照してください。

[戦闘画面]
敵に遭遇すると戦闘になります。コマンドは6つ。
  ⚔ たたかう   … 武器で攻撃する
  ✨ まほう     … 覚えたまほうを使う(MPを消費)
  🧪 アイテム   … やくそう(回復)や捕獲ロープ(捕獲率UP)を使う
  🪤 つかまえる … 敵を捕獲する。HPを減らすほど成功しやすい
  🎵 うたう     … 特殊コマンド。通常敵では次の捕獲率を少し上げる
                  究極ゴリラには特別な効果がある(後述)
  💨 にげる     … 戦闘から逃げる

・戦闘の「🧪アイテム」から回復アイテム(コーヒー/パン/お弁当/ラーメン)を
  使えます。HPが少ない時に使いましょう。HP満タン時は消費されません。

・このゲームは「倒すゲーム」ではなく、UMAを弱らせて捕まえる/
  逃げられるUMA収集RPGです。
・敵のHPを0にすると「○○に逃げられた！」という演出になり、
  経験値と小額のお金が手に入ります。
・捕まえる・逃げる・逃げられる、いずれの結果でも経験値が入ります
  (逃げた場合は少なめです)。
・HPを1〜10程度まで減らすと、通常の敵はかなり捕まえやすくなります。
  捕獲ロープを使うとさらに捕獲率が上がります。
  レアUMAはHPを減らしても難しく、究極ゴリラはほぼ捕獲できません。
・低確率でメタルゴリラ系(メタルゴリラ/はぐれメタルゴリラ/
  フルメタルゴリラ)が出現します。防御が非常に高く攻撃はほぼ通りませんが、
  倒すと非常に多い経験値が手に入ります(Lv99を目指す上での重要な稼ぎ相手！)。
  遭遇時は「キラリと光った！経験値のチャンスだ！」というメッセージが出ます。
・経験値が一定値を超えるとレベルアップします。
・レベルアップ時、ランダムで新しい「まほう」を1つ覚えます。
  (HP/MPの上限やこうげき力・ぼうぎょ力も上がります)
・HPが0になっても村に戻ってやり直せます(ペナルティは軽めです)。
・一部の敵の攻撃を受けると、低確率で状態異常になることがあります。
    🤧 アレルギー … フィールドを歩くたびにHPが少し減る(12歩で自然に治る)
    👃 におい     … UMAの捕獲成功率が下がる(3回戦闘すると自然に治る)
  状態異常になっている間は、画面左上のステータス欄にアイコンと
  名前が表示されます。戦闘中に「🧪アイテム」からせき止めシロップ
  (アレルギーを治療)/デオドラントスプレー(においを治療)を使うと
  すぐに治せます(商人で購入できます)。該当する状態異常でない時に
  使うと「今は使う必要がない」と表示され、消費されません。

[UMA図鑑]
・画面右上の「📖図鑑」ボタンから、これまでに遭遇したUMAの
  一覧を確認できます。3段階で表示されます。
    ？？？        … 未発見(一度も遭遇していない)
    UMA名 発見済   … 戦闘で遭遇したが、まだ捕獲していない
    UMA名 捕獲済   … 一度でも捕獲に成功した
  捕獲済みのUMAを後で売っても、図鑑の「捕獲済み」記録は消えません。
・発見済みまたは捕獲済みのUMAをタップ(クリック)すると、
  そのUMAの詳細ステータス(HP/攻撃力/防御力/捕獲率/経験値/説明など)
  を確認できます。
・究極ゴリラは特別表示：捕獲前は「普通の捕獲は通用しない」警告、
  捕獲後は金枠と「伝説のUMA」「森へ帰った」バッジが表示されます。
・図鑑の下部には「メタル系（特殊エネミー）」セクションがあり、
  メタルゴリラ系3種の詳細をタップで確認できます。
・図鑑一覧の上部に「発見 N/9」「捕獲 N/9」の進捗が表示されます。

[レアUMAと最終目標]
・ビッグフット、ネッシー、イエティ、ジャージーデビルなどの
  レアUMAは、レベルに関係なくどこでも低確率で出現します。
  かなり強いので、勝てそうにない時は「にげる」を選んでください
  (レアUMAは比較的逃げやすく調整されています)。
・そして、ごく低確率で伝説のUMA「究極ゴリラ」🦍 が出現します。
  これを捕まえることが、このゲームの最終目標です！
  発見した瞬間に専用のメッセージが表示されます。

[究極ゴリラについて(重要)]
・究極ゴリラは完全にラスボス級の強さで、レベルが低いうちはまず
  間違いなく一撃で倒されます。攻撃はすべて「かいしんのいちげき」級の
  大ダメージで、ぼうぎょ力で受け止めることはほぼできません。
・通常の「つかまえる」コマンドは究極ゴリラには通用しません。
  特別な方法でしか捕まえられません。
・低レベルでも出会うこと自体は普通にあります。出会ったら戦う必要は
  なく、「にげる」を選べばほぼ確実に逃げられます(究極ゴリラは
  あえて逃げやすく調整されています)。倒れてしまっても村に戻って
  やり直せるので、安心して遭遇を楽しんでください。

[究極ゴリラの捕まえ方]
究極ゴリラを捕まえるには、3つの条件をすべて満たす必要があります。

  条件1: レベル99まで成長していること
  条件2: 「女神のウクレレ」🪗 を所持していること
          (マップ上の特別な宝箱🪗から入手できます)
  条件3: 究極ゴリラのHPを1〜10まで削ること

3つの条件が揃ったら、戦闘中の「🎵 うたう」コマンドを使ってください。
勇者の子孫が女神のウクレレを奏で、究極ゴリラの怒りが静まります。
成功するとエンディングが始まります！

[エンディングについて]
・究極ゴリラを捕まえると、5ページ構成のエンディングが表示されます。
  「つぎへ ▶」でページを送り、最後に「冒険を続ける」で終了します。
・クリア後も冒険は続けられます。図鑑集め・装備集め・仲間集めをお楽しみください。
・エンディングは設定画面(⚙️)またはステータス画面(📊)からいつでも
  再視聴できます(「🎬 エンディングを見る」ボタンを選んでください)。
・クリア後はステータス画面に称号「森に歌を届けし者」が表示されます。
・ニューゲームを行うとクリア状態もリセットされます。


----------------------------------------------------
■ 仲間について(フィールドの🍺酒場に触れる)
----------------------------------------------------
村の 🍺(酒場)に触れると酒場メニューが開きます。

  🤝 仲間を探す … 仲間候補に声をかけて加入を試みる
  👥 仲間を見る … 現在の仲間と効果を確認する
  👋 仲間を外す … 仲間をパーティーから外す
  やめる         … 閉じる

[仲間候補と効果]
  💪 ジュリタニ … 攻撃時に低確率で「会心の一撃」(ダメージ1.5倍)
  🪤 シュリタニ … UMAの捕獲率+10%
  💨 ノリオ     … 逃走成功率+15%
  ✨ ハルミ     … まほうの効果(ダメージ/回復量)+20%

・加入を試みると、仲間候補が固有の反応を見せます。加入率は人によって異なり、
  断られることもあります。断られても候補は酒場に残るので何度でも再挑戦できます。
・仲間は最大2人まで連れていけます。上限の場合は「仲間を外す」で調整してください。
・「仲間を外す」で外した仲間は再度「仲間を探す」から加入できます。
・仲間情報はセーブデータに保存されます。ニューゲームでリセットされます。
・仲間はUMAではなく、売却できません。

■ 商人の使い方(フィールドの🏪に触れる)
----------------------------------------------------
村の中にある 🏪(商人)に歩いて触れると、売買メニューが開きます。
  🛍️ 買う          … やくそう・捕獲ロープ・コーヒー(5G)/パン(10G)/
                      お弁当(20G)/ラーメン(40G)/せき止めシロップ(15G)/
                      デオドラントスプレー(15G)・鉄の剣を所持金で購入
  ⚔ 武器を買う      … ワイヤーブラシ/ノコギリ/魔法のステッキ等 多数
  🥋 防具を買う      … ロックT/革ジャン
  🛡 盾を買う        … 鉄のたて
  ⛑ 兜を買う         … ヘルメット
  📤 アイテムを売る … 持っているやくそう/捕獲ロープを売ってお金にする
  🦍 UMAを売る      … 捕獲して所持しているUMAを売ってお金にする
  🔧 装備を売る     … 所持している装備を購入価格の半額で売る
                      (現在装備中のもの・★伝説装備は売れません)
  やめる            … 商人メニューを閉じる

・買い物のお金は画面左上の「💰 ○○G」(所持金)で確認できます。
・鉄の剣を買うと、その場でこうげき力が上がります
 (装備の付け外しはなく、買った瞬間に効果が反映される簡易仕様です)。
・武器/防具/盾/兜を買うと所持品に追加されます。買った時点では
  まだ装備されないので、画面右上「🎽装備」の装備変更画面で
  改めて装備してください。
・お金が足りない時は購入できません(トーストで知らせます)。

[伝説装備について]
・最強クラスの装備5種は商人では売っていません。フィールドの特別な
  宝箱か実家のイベントで入手します。
  🌟 ペガサスのよろい(防御+14 HP+5): フィールドの白く光る宝箱。Lv50以上が必要。
  六連のたて(防御+20): Lv60以上で実家に帰って「休む」と見つかる。
  ⭐ 宇宙のかぶと(防御+15): フィールドの星のような宝箱。女神のウクレレが必要。
  🪄 如意棒(攻撃+58): フィールドの岩に刺さった棒。Lv70以上＋ジュリタニが必要。
  アンドロメダの鎖(攻撃+44): クリア後に実家で「休む」と王様の使者が届けてくれる。
・伝説装備は売ることができません。
・ステータス画面(📊)で入手進捗(N/5)を確認できます。

[UMA売却について]
・「UMAを売る」では、今持っているUMAの種類と数、売却額が一覧表示
  されます。売ると所持数が1減り、お金が増えます。
・同じUMAを複数捕まえていれば、何匹でも売れます。
・UMAを売り切っても、UMA図鑑の「捕獲済み」記録はそのまま残ります
 (図鑑は「捕まえた実績」、所持UMAは「今持っている数」という扱いです)。


----------------------------------------------------
■ 転職の説明(フィールドの⛩️に触れる)
----------------------------------------------------
村の中にある ⛩️(神様の社)に歩いて触れると、転職メニューが開きます。
なりたい部活(職業)のボタンを選ぶと、その場で転職します。
お金や条件は不要で、いつでも何度でも転職し直せます。
現在の職業は画面左上のステータス欄に「(○○部)」と表示されます。

[職業ごとの特徴]
  野球部     … こうげき力が少し高い
  水泳部     … 最大HPが少し高い
  テニス部   … UMAを捕まえやすい
  帰宅部     … 戦闘から逃げやすい
  吹奏楽部   … まほうを覚えやすい(レベルアップ時に追加でまほうを覚えることがある)
  サッカー部 … 全部少しずつ強い、バランス型(最初の職業)
  ラグビー部 … HPとこうげき力が高いが、その分逃げにくい
  陸上部     … とても逃げやすい
  魔法戦士   … まほうとこうげきのバランス型(MPも少し増える)

・転職すると、こうげき力・ぼうぎょ力・最大HP・最大MPがその場で
  再計算されます。最大値が下がる転職をした場合、現在のHP/MPが
  超えていれば新しい上限まで自動的に下がります(全回復はしません)。
・捕獲率/逃走率/まほうの覚えやすさへの補正は、戦闘中の判定にだけ
  反映されます(見た目のステータス欄には数値で出ません)。

■ 設定画面(画面右上「⚙️設定」)
----------------------------------------------------
画面右上の「⚙️設定」ボタンから設定画面を開けます。

・主人公の歩く速さを変更できます: 遅い / 普通(初期値) / 速い
  十字キーを押しっぱなしにすると、選んだ速さで連続移動します。
  軽くタップした場合は、これまでと同じく1回のタップで1歩だけ
  動きます(タップ操作の感覚は変わりません)。
・「💾オートセーブ中」の説明と、「💾今すぐセーブ」ボタンがあります。
  押すとその場でセーブし、「セーブしました」と表示されます。
・「🎯目的を見る」で、初回起動時に表示された目的説明をいつでも
  見直せます。
・「❓ヘルプ」で、目的/操作方法/セーブ/戦闘/捕獲/装備/状態異常を
  まとめて確認できます。
・「🔄ニューゲーム(セーブデータをリセット)」ボタンで最初からやり
  直せます。確認ダイアログが出てから削除されます。
・設定はセーブデータと一緒にブラウザに保存されます。

■ ステータス確認画面(画面右上「📊ステータス」)
----------------------------------------------------
画面右上の「📊ステータス」ボタンから、現在のプレイヤー情報を
まとめて確認できます。

  現在の目標(進行状態に応じて自動表示)/
  称号(クリア前「勇者の子孫」/クリア後「森に歌を届けし者」)/
  名前/レベル/HP・MP/経験値/所持金/職業/状態異常/
  現在の装備(武器・防具・盾・兜)/覚えているまほう/
  所持アイテム数(やくそう・捕獲ロープ・せき止めシロップ・
  デオドラントスプレー)/捕まえたUMA総数/UMA図鑑の進捗/
  重要アイテム(女神のウクレレ)/クリア記録


----------------------------------------------------
■ 装備変更画面(画面右上「🎽装備」)
----------------------------------------------------
画面右上の「🎽装備」ボタンから装備変更画面を開けます。
武器・防具・盾・兜の4種類のスロットがあり、それぞれ1つだけ装備できます。

・各カテゴリの一覧から、自分が「所持している」項目を選んで
  「装備する」を押すと、その場で装備が切り替わり、こうげき力・
  ぼうぎょ力(一部はHP/MPも)が再計算されます。
・所持していない装備は「未所持」と表示され、選べません。
  商人(🏪)で購入すると所持品に追加されます(■商人の使い方を参照)。
・木の棒/Tシャツ/段ボールのたて/男塾ハチマキはボーナスのない
  初期装備で、最初から所持・装備しています。
・以前のバージョン(装備を自由に選べた頃)からの引っ越しプレイヤーは、
  その時点で装備していたものを自動的に所持品として引き継いでいます。


----------------------------------------------------
■ セーブ/ロードについて
----------------------------------------------------
ブラウザの localStorage を使って自動セーブしています。
アイテム取得・戦闘終了・買い物・転職などのたびに自動で保存され、
次回 index.html を開いたときに自動で読み込まれます
(読み込まれた場合は「💾 前回のデータを読み込みました」と表示されます)。

  保存される内容: レベル/経験値/HP・MP/覚えたまほう/所持金/
  所持アイテム数(やくそう・捕獲ロープ・せき止めシロップ・
  デオドラントスプレー)/所持UMA/UMA図鑑の状態/現在の職業/歩く速度/
  装備中の武器・防具・盾・兜/所持している武器・防具・盾・兜/
  状態異常/オープニング・目的説明の既読フラグ/
  女神のウクレレ所持状態/ゲームクリアフラグ

  保存されない内容(TODO): フィールド上の現在位置、拾った/拾って
  いないフィールドアイテムの状態。再読み込み時は村の入口から
  スタートします。必要であれば script.js の saveGame()/loadGame()
  に state.player.x, y や state.items の差分を追加してください。

  ※ localStorageはブラウザ・端末ごとに保存されるため、別のPCや
    別のブラウザ、別の場所に展開したフォルダには引き継がれません。
  ※ ブラウザの設定でサイトデータ/履歴を消去すると、セーブデータも
    一緒に消えます。

■ 開発者向けメモ: データの編集場所
----------------------------------------------------
※ ゲーム仕様の正式なドキュメントは GAME_DESIGN.md、実装予定は TODO.md、
  変更履歴は CHANGELOG.md にまとめてあります。新しい仕様はまず
  GAME_DESIGN.md に整理してから実装する運用にしています。

あばれうまのりお からのアイデア追加など、今後のネタ出しに対応
しやすいよう、script.js の冒頭(「2. データ定義」のコメント以降)に
データをまとめています。基本的にロジック部分(関数)を触らずに、
配列へ要素を追加・編集するだけで内容を拡張できます。

  UMA_DATA          … 捕獲・図鑑登録・売却の対象になるUMA本体のデータ
                       (id/name/rarity/hp/attack/captureRate/exp/
                        sellPrice/isRareなど。inflictsで状態異常も付与可)
  NON_UMA_DATA      … UMAではない雑魚モンスター(スライム等。図鑑対象外)
  ITEM_DATA         … やくそう・捕獲ロープなどの消耗品(価格・効果)。
                       trackable:true のものだけ実際に売買・所持できる
  WEAPON_DATA       … レガシー武器(つるぎ/鉄の剣。購入/取得で即時加算)
  EQUIP_WEAPON_DATA … 装備変更画面用の武器一覧(§8.5)。buyPriceがある
                       項目だけ商人の「武器を買う」に表示される
  ARMOR_DATA        … 装備変更画面用の防具一覧(同上、buyPriceで購入可否)
  SHIELD_DATA       … 装備変更画面用の盾一覧(同上)
  HELMET_DATA       … 装備変更画面用の兜一覧(同上)
  ATTACK_SPELL_DATA … 攻撃まほう  HEAL_SPELL_DATA … 回復まほう
                       (SPELL_DATAは両方を結合した互換用配列)
  JOB_DATA          … 職業(部活)ごとのステータス補正と説明文

例えば新しいUMAを1匹追加したい場合は、UMA_DATA配列に1行追加する
だけで、図鑑・エンカウント・捕獲・売却すべてに自動的に反映されます。

究極ゴリラの出現を素早く確認したい場合は、script.js の
「3. エンカウント関連の調整値」付近にある下記の定数を調整してください。

  var RARE_ENCOUNTER_CHANCE = 0.08;  // ここを上げるとレアUMA全体が出やすくなる

さらに UMA_DATA 配列の中の
  { id: "ultimategorilla", ... weight: 4 ... }
の weight の値を大きくすると、レアUMAの中でも究極ゴリラが
選ばれやすくなります。

マップ(RAW_MAP)を編集して、商人(M)や神様(G)の位置を増やしたり、
新しいエリアを追加することもできます。1行=1文字につき1マスです。

  処理の前段に、課金導線を差し込めるような分岐を用意すると良い。

  実際の課金処理はWebブラウザの素のJavaScriptだけでは完結しない
  (ネイティブアプリ化やストア決済SDKの導入が前提になる)。
